# Houston AQI Predictor

An interactive Streamlit app that predicts next-day air quality (AQI) for Houston using machine learning on historical AQI, weather, and trend features.

## Features
- Next-day AQI prediction
- Streamlit dashboard
- Random Forest classifier
- Feature engineering pipeline
- Weather + pollution analysis

## Tech Stack
- Python
- Scikit-learn
- Streamlit
- Pandas
- NumPy

## Models Used
- Logistic Regression
- Random Forest
- LightGBM
